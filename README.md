# Admintool
### Software:
* ```ubuntu /14.04.1```
* ```nginx /1.4.6```

### Dependecies:
* ```jQuery /2.2.2```
* ```bootstrap /3.3.6```

# Developer Installation

```
npm install
```

And now, start developing:

```
npm start
```

### File Structure:
* admin (Admintool)
  * res (Resources)
    * bg (big pictures and backgrounds)
    * favicon
    * fonts
    * logo
    * bootstrap
    * js
  * src
* public (Website)
